# Cannes Metal Market

Minimal agency / office exchange built from a Cannes Lions 2026 winner dataset, collected via a Google-powered scraping workflow by Miguel Castro (https://miguelcastro.works/) and the Day 2 winners update.

## Rules

- Metal winners only: Grand Prix, Gold, Silver, Bronze.
- No shortlist or submitted rows.
- No network rollups when office/city/country detail is available.
- TBWA office names use slash format: `TBWA / Canada Toronto`, not `TBWA\Canada Toronto`.
- Creative Brand company-led winners are excluded from the agency exchange.

## Value formula

```text
Base Value = GP×100 + Gold×40 + Silver×15 + Bronze×5
E = round(Base Value ÷ entered cases), capped at +100 pts
Value = Base Value + E
```
